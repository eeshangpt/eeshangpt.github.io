# Personal Website

A modern, single-page personal portfolio site for **Eeshan Gupta (Senior AI Engineer)**.

The project is a lightweight static website built with plain HTML, CSS, and JavaScript. It highlights experience, selected AI projects, skills, consulting offerings, and contact information.

## Tech Stack

- HTML5
- CSS3 (custom properties, responsive grid layouts, transitions)
- Vanilla JavaScript (intersection observer reveals and benchmark bar animation)
- Google Fonts (`Space Grotesk`, `Inter`)

## Project Structure

```text
.
├── index.html    # Page content and section structure
├── styles.css    # Theme, layout, responsiveness, and component styles
├── script.js     # Scroll reveal + hero benchmark animation
└── README.md
```
